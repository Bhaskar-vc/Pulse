Pagination

Pagination allows users to navigate through large datasets split into pages. Supports numbered pages, range counters, rows-per-page selectors, and quick-jump inputs.

Stable Accessible Figma Ready

Overview

The three core variants from the design system: Short (compact range counter), Default (with rows-per-page and go-to), and numbered page buttons.

CSS Classes

ClassDescription
.pgnNumbered pagination container (base)
.pgn--smSmall size — 28px buttons
.pgn--lgLarge size — 44px buttons
.pgn--roundedFully rounded pill-shaped buttons
.pgn--outlineOutlined active state (no fill)
.pgn--ghostMinimal / ghost buttons, no border
.pgn-btnIndividual page or nav button
.pgn-btn.is-activeCurrently active page state
.pgn-pagesWrapper for numbered page buttons
.pgn-ellipsisNon-interactive ellipsis separator
.pgn-shortCompact range counter with arrows
.pgn-fullFull variant with rows-per-page, go-to
.pgn-rpp-selectStyled native select for rows per page
.pgn-inputGo-to page text input
.pgn-arrowSimple icon-only arrow button (short variant)

Code

Short variant

pagination.html 
<!-- Short: range counter + arrows -->
<div class="pgn-short">
  <div class="pgn-counter">
    <span>1</span><span>–</span><span>10</span>
    <span class="dim">of</span>
    <span class="dim">35</span>
  </div>
  <div class="pgn-nav">
    <button class="pgn-arrow" disabled>←</button>
    <button class="pgn-arrow">→</button>
  </div>
</div>

Numbered variant

pagination.html 
<nav class="pgn" aria-label="Page navigation">
  <button class="pgn-btn" disabled aria-label="Previous page">‹</button>
  <div class="pgn-pages">
    <button class="pgn-btn is-active" aria-current="page">1</button>
    <button class="pgn-btn">2</button>
    <button class="pgn-btn">3</button>
    <span class="pgn-ellipsis">···</span>
    <button class="pgn-btn">9</button>
    <button class="pgn-btn">10</button>
  </div>
  <button class="pgn-btn" aria-label="Next page">›</button>
</nav>
