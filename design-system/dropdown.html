Dropdown

Dropdowns present a list of options the user can select from. Use them for single selection, grouped items, searchable lists, and action menus.

Stable Accessible Figma Ready

Overview

Click the trigger to open the menu. Arrow keys navigate items, Enter/Space selects, Escape closes. Click outside also closes.

CSS Classes

ClassDescription
.dd-wrapPosition: relative container — scopes the floating menu
.dd-triggerButton-style trigger — full width, 40px height
.dd-trigger.is-openActive / focused state — purple border + ring
.dd-trigger.is-disabledDisabled state — grey background, no events
.dd-trigger.is-errorError state — red border
.dd-menuFloating options list — absolutely positioned below trigger
.dd-menu.is-openVisible state — opacity 1, pointer events enabled
.dd-itemSingle option row — 9px 14px padding, hover grey
.dd-item.is-selectedActive selection — purple text + checkmark
.dd-item.is-disabledNon-interactive option — grey text
.dd-item.is-dangerDestructive action — red text, red hover background
.dd-group-labelSection heading inside the menu
.dd-divider1px horizontal separator between groups
.dd-searchText input for filtering options inline
.dd-item-icon18px icon container before item label
.dd-item-avatar24px circular initials avatar
.dd-item-badgePill badge pushed to the right inside an item

Code

dropdown.html
<div class="dd-wrap">
  <button class="dd-trigger"
          aria-haspopup="listbox"
          aria-expanded="false"
          onclick="toggleDd('my-dd')">
    <span class="dd-placeholder">Select option</span>
    <!-- chevron SVG -->
  </button>

  <div class="dd-menu" id="my-dd-menu" role="listbox">
    <div class="dd-item" role="option">Option A</div>
    <div class="dd-item is-selected" role="option" aria-selected="true">Option B</div>
    <div class="dd-item is-disabled" role="option" aria-disabled="true">Option C</div>
  </div>
</div>
