Steps / Progress

Step indicators guide users through multi-step workflows — onboarding, checkout, setup wizards. Each step shows completion state: done, active, or upcoming.

Stable Accessible Figma Ready

Overview

The default steps component shows numbered nodes connected by a line. Completed steps use a checkmark, the active step has a purple ring, and upcoming steps are grey.

ClassDescription
.stepsHorizontal flex container — distributes steps evenly
.steps--sm28px nodes, smaller text
.steps--lg44px nodes, larger text
.steps--dot12px dot nodes — minimal progress indicator
.steps--verticalStacks steps in a column with a vertical connector line
.stepIndividual step — flex column, centered
.step.is-doneCompleted — purple filled node, purple connector to next
.step.is-activeCurrent step — purple ring, purple step title
.step.is-errorFailed step — red node with warning icon
.step-node36px circle — number or icon
.step-textLabel container below the node
.step-title13px / semibold step label
.step-desc12px muted supporting text
