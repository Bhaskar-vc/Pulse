Rating

A star rating component for collecting or displaying user feedback. Supports full and half stars, multiple sizes, color variants, read-only and interactive modes, and accessible keyboard navigation.

Stable WCAG AA Figma 365-104529

Overview

Click the stars below to set a rating. The component supports half-step values through the clip-path technique.

4.0
Click a star to rate

Display Modes

Rating can be used in read-only mode to display scores or in interactive mode for user input. Half-stars are rendered using a CSS clip-path technique on a layered SVG.

Read-only — All Values (0 to 5, step 0.5)
Interactive — Hover Highlighting + Click to Set
5-star interactive
No rating selected
3-star interactive
No rating selected

Item Count Variants

The max prop controls how many stars are shown. Common values are 3, 4, and 5.

Triple (3) max=3, value=2.5
Quadruple (4) max=4, value=3
Quintuple (5) max=5, value=4.5 (default)

With Count Label

Use showCount with a countLabel string to display the numeric score and review count alongside the stars.

Sizes

Three size options are available via the size prop: sm, md (default), and lg.

sm 16 × 16px stars
md 20 × 20px stars · default
lg 28 × 28px stars

Color Variants

Use color to change the filled-star color. Gold is the default for ratings; purple and gray suit branded contexts or read-only indicators.

Gold Default · #F79009 · ratings, reviews
Purple Brand accent · #7f56d9 · interactive preference
Gray Neutral · #9d9dad · read-only, disabled state

Props

Prop Type Default Description
value number 0 Current rating value. Supports 0–5 in 0.5 increments.
max 3 | 4 | 5 5 Total number of stars (items). "triple", "quadruple", "quintuple" in Figma.
size 'sm' | 'md' | 'lg' 'md' Star size. sm = 16px, md = 20px, lg = 28px.
readonly boolean false When true, disables hover and click interaction. Renders in display mode.
showCount boolean false When true, renders a count label next to the stars.
countLabel string — Label string shown when showCount is true. E.g. "4.5 (128 reviews)".
color 'gold' | 'purple' | 'gray' 'gold' Fill color of the selected stars.
customIcon boolean | SVGElement false When false, uses the default gold star. When true, accepts a custom SVG path.
onChange (value: number) => void — Callback fired when user selects a rating. Not used in readonly mode.

Do's & Don'ts

Do
  • Use gold stars for ratings that reflect quality or satisfaction.
  • Always include a count label when showing aggregate scores (e.g. "4.5 (128 reviews)").
  • Use readonly to display pre-existing ratings — not interactive.
  • Use aria-label on the radiogroup to describe what is being rated.
  • Support keyboard navigation (arrow keys to adjust, Enter/Space to confirm).
  • Maintain consistent star counts across the same page or context.
Don't
  • Don't use different star counts (3, 4, 5) for the same rating dimension on one page.
  • Don't use star ratings as a primary navigation element or button.
  • Don't place interactive stars inside disabled form fields without setting readonly.
  • Don't omit the aria-label — screen readers need context for what is being rated.
  • Don't rely solely on color to convey the rating — always pair with numeric value or label.
  • Don't show partial half-stars for values already at a full integer (e.g. 3.0).

Accessibility

The Rating component is built on the ARIA radiogroup pattern. Each star is a radio input that screen readers can announce individually. Keyboard users can tab into the group and use arrow keys to adjust the value.

ARIA Roles

  • Wrapper: role="radiogroup"
  • Each star: role="radio"
  • Wrapper: aria-label="Rate this product"
  • Selected star: aria-checked="true"
  • Others: aria-checked="false"

Keyboard Interaction

  • Tab — move focus into the rating group
  • Arrow Right / Up — increase rating by 1
  • Arrow Left / Down — decrease rating by 1
  • Home — set rating to minimum (0 or 1)
  • End — set rating to maximum

Screen Reader

  • Read-only: announce as "4.5 out of 5 stars"
  • Interactive: announce each option "1 star", "2 stars"…
  • Selected: prefix "Selected, 3 stars, 3 of 5"
  • Include count label in aria-describedby if present

Color & Contrast

  • Gold (#F79009) on white: 3.0:1 — use with text label
  • Always pair color with numeric value for clarity
  • Empty star stroke (#eaeaed) must remain visible on white
  • Focus ring must meet 3:1 contrast ratio

Half-star CSS technique
CSS 
/* Half-star: stack two SVGs, clip the foreground to 50% */
.star-half-wrap {
  position: relative;
  display: inline-flex;
}
.star-half-wrap .star-bg {
  position: absolute;
  top: 0; left: 0;
}
.star-half-wrap .star-fg {
  position: relative;
  clip-path: inset(0 50% 0 0);  /* clips right half */
}

/* Star SVG path */
.star-empty { fill: none; stroke: #eaeaed; stroke-width: 1.5; }
.star-filled { fill: #F79009; stroke: #F79009; stroke-width: 0.5; }
JavaScript — render stars from value
JavaScript 
// Star SVG path constant
const STAR_PATH = "M12 2l3.09 6.26L22 9.27l-5 4.87 1.18 6.88L12 17.77l-6.18 3.25L7 14.14 2 9.27l6.91-1.01L12 2z";

function renderRating(container, { value = 0, max = 5, size = 'md', color = 'gold' }) {
  const sz = { sm: 16, md: 20, lg: 28 }[size] || 20;
  const fillClass = { gold: 'star-filled', purple: 'star-filled-purple', gray: 'star-filled-gray' }[color];
  container.innerHTML = '';

  for (let i = 1; i <= max; i++) {
    const filled = value >= i;
    const half   = !filled && value >= i - 0.5;

    const wrap = document.createElement('span');
    wrap.className = 'star-wrap';

    if (half) {
      wrap.innerHTML = `<span class="star-half-wrap">
        <svg class="star-bg" viewBox="0 0 24 24" width="${sz}" height="${sz}">
          <path class="star-empty" d="${STAR_PATH}"/>
        </svg>
        <svg class="star-fg" viewBox="0 0 24 24" width="${sz}" height="${sz}">
          <path class="${fillClass}" d="${STAR_PATH}"/>
        </svg>
      </span>`;
    } else {
      const cls = filled ? fillClass : 'star-empty';
      wrap.innerHTML = `<svg viewBox="0 0 24 24" width="${sz}" height="${sz}"><path class="${cls}" d="${STAR_PATH}"/></svg>`;
    }
    container.appendChild(wrap);
  }
}
