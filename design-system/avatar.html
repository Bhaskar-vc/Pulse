Avatar

Avatars represent users or entities with a photo, initials, or placeholder icon. Available in six sizes with optional status indicators (online, company), group stacking, and label groups.

Props / API

HTML/CSS class-based API. Compose classes to build any avatar variant.

Class Values Description
.av base class Required on every avatar element. Sets circular shape and flex layout.
Size modifier .av-2xs .av-xs .av-sm .av-md .av-lg .av-xl .av-2xl Sets width/height to 20 / 24 / 32 / 40 / 48 / 56 / 64 px respectively.
Color palette .av-purple .av-blue .av-green .av-orange .av-teal .av-pink .av-gray Background + text color for initial-based avatars. Use when no photo is available.
.av-placeholder modifier Light purple background for icon placeholder. Place an SVG icon inside.
.av-wrap container Wraps .av to enable absolute-positioned status badges. Add a size class (e.g. .av-wrap.av-md) to size the badge correctly.
.av-online badge (inside .av-wrap) Green online status indicator, bottom-right corner. Scales with the size class on the parent .av-wrap.
.av-company badge (inside .av-wrap) Company logo icon badge, bottom-right corner. Place an <img> or .av-company-inner inside.
.av-ring wrapper modifier Adds a light purple focus/hover ring by setting a border: 4px solid #f4ebff on the inner .av.
.av-group container Stacks avatars with overlap via negative left margin. Add a size class (.av-xs/.av-sm/.av-md) to control gap.
.av-more child of .av-group Overflow counter avatar. Also needs a size class. Shows "+N" in brand purple on a light background.
.av-add standalone or child of .av-group Dashed-border "Add user" button. Combine with a size class (.av-xs/.av-sm/.av-md).
.av-label-group container Horizontal flex row combining an avatar with a name/email text block. Add .size-sm/.size-md/.size-lg/.size-xl for spacing.
.av-profile standalone Large profile photo variant with 4px white border and drop shadow. Use with .av-profile-md (96 px) or .av-profile-lg (160 px).

Usage

<!-- Basic initials avatar -->
<span class="av av-md av-purple">OR</span>

<!-- Avatar with photo -->
<span class="av av-md">
  <img src="olivia.jpg" alt="Olivia Rhye"/>
</span>

<!-- With online indicator -->
<span class="av-wrap av-md">
  <span class="av av-md av-purple">OR</span>
  <span class="av-online"></span>
</span>

<!-- Avatar group with overflow + add button -->
<div class="av-group av-sm">
  <span class="av av-sm av-purple">OR</span>
  <span class="av av-sm av-blue">PB</span>
  <span class="av av-sm av-more">+3</span>
</div>
<button class="av-add av-sm">+</button>

<!-- Label group -->
<div class="av-label-group size-md">
  <span class="av av-md av-purple">OR</span>
  <div class="av-label-group-text">
    <span class="av-label-name">Olivia Rhye</span>
    <span class="av-label-email">olivia@untitledui.com</span>
  </div>
</div>
