Segment

A segmented control lets users select one option from a small set of mutually exclusive choices. It presents all options in a compact inline group, making the current selection immediately visible.

Usage guidelines

When to use segments and how to avoid common mistakes.

✓ Do

Use 2–5 options. Keep labels short (one or two words). Use when all options are visible at once and the selection is mutually exclusive.

✕ Don't

Don't use more than 7 options — use a select dropdown instead. Avoid long labels; they break the compact layout. Don't use for multi-select scenarios.

✓ Do

Use segments to switch views, filter content, or toggle between modes. Pair with related content that changes immediately on selection.

✕ Don't

Don't confuse with tabs — tabs navigate between distinct page sections, while segments filter or change a view inline without navigating.

Accessibility

Segment controls follow the ARIA radio group pattern.

Keyboard interaction

  • Tab — moves focus into/out of the segment group
  • Arrow Left / Right — cycles between options within the group
  • Space / Enter — selects the focused option

ARIA pattern

  • Container uses role="radiogroup"
  • Each item uses role="radio"
  • Selected item has aria-checked="true"
  • Provide aria-label on the container

Focus management

  • Focus ring uses box-shadow for visibility
  • Only the active item is in the Tab sequence
  • Arrow keys move within the group

Icon-only segments

  • Must include aria-label on each button
  • Tooltip is recommended but not required
  • Ensure sufficient touch target size (min 40×40px)

Code

Copy-ready HTML and CSS for the default segment control.

segment.html 
<!-- Segment control -->
<div class="segment"
     role="radiogroup"
     aria-label="View mode">

  <button class="segment-item is-active"
          role="radio"
          aria-checked="true">Monthly</button>

  <button class="segment-item"
          role="radio"
          aria-checked="false">Quarterly</button>

  <button class="segment-item"
          role="radio"
          aria-checked="false">Annually</button>

</div>

segment.js — interactive selection 
// Make all .segment groups interactive
document.querySelectorAll('.segment').forEach(group => {
  group.addEventListener('click', e => {
    const btn = e.target.closest('.segment-item');
    if (!btn || btn.disabled) return;

    // Deactivate all
    group.querySelectorAll('.segment-item').forEach(item => {
      item.classList.remove('is-active');
      item.setAttribute('aria-checked', 'false');
    });

    // Activate clicked
    btn.classList.add('is-active');
    btn.setAttribute('aria-checked', 'true');
  });

  // Arrow key navigation
  group.addEventListener('keydown', e => {
    const items = [...group.querySelectorAll('.segment-item:not([disabled])')];
    const idx = items.indexOf(document.activeElement);
    if (e.key === 'ArrowRight') items[(idx + 1) % items.length].focus();
    if (e.key === 'ArrowLeft')  items[(idx - 1 + items.length) % items.length].focus();
  });
});
