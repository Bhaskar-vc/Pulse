LABEL

Label

Form field labels that communicate the purpose of an input. Supports three visual types and three modifiers — required asterisk, optional indicator, and tooltip — that can be combined freely.

Stable WCAG AA Figma · 1270-41013

Overview

Three types cover most label needs. Default and Light are visually identical (12 px, muted) and used for standard form fields. Large is 14 px medium weight for prominent headings above grouped fields.

In Context

Labels are placed directly above their associated input with a 6 px gap. Pair with form inputs, textareas, and select controls.

Props / API

Class-based API. Compose classes to build any label variant.

Element / Class Values / Notes Description
.lbl Wrapper Required on every label. Inline-flex row, gap:2px, height:20px. Use as a <label> element for accessibility.
.lbl-text Type: Default / Light 12 px · Regular · #545470. Default and Light types are visually identical.
.lbl-text.lbl-large Type: Large 14 px · Medium (500) · #29294c. Use for prominent grouped field headings.
.lbl-required Modifier Red asterisk *. 14 px · Regular · #d92d20. Place immediately after .lbl-text.
.lbl-optional Modifier Muted (optional) text. 12 px · Regular · #707087. Mutually exclusive with .lbl-required.
.lbl-tooltip Modifier Info icon wrapper. 12 × 12 px. Hover reveals a dark tooltip popup. Contains a .lbl-tooltip-popup child for the tooltip text.
.lbl-tooltip-popup Child of .lbl-tooltip Hidden by default, shown on :hover. Renders a dark chip above the icon with an arrow pointer.

Usage

<!-- Default label (plain) -->
<label class="lbl" for="field-id">
  <span class="lbl-text">First name</span>
</label>

<!-- Large · required -->
<label class="lbl" for="email">
  <span class="lbl-text lbl-large">Email address</span>
  <span class="lbl-required">*</span>
</label>

<!-- Default · optional -->
<label class="lbl" for="company">
  <span class="lbl-text">Company</span>
  <span class="lbl-optional">(optional)</span>
</label>

<!-- Large · required + tooltip -->
<label class="lbl" for="vat">
  <span class="lbl-text lbl-large">VAT number</span>
  <span class="lbl-required">*</span>
  <span class="lbl-tooltip">
    <!-- info SVG icon -->
    <svg width="12" height="12" viewBox="0 0 12 12" fill="none">
      <circle cx="6" cy="6" r="5.25" stroke="currentColor" stroke-width="1.5"/>
      <path d="M6 5.5v3.5" stroke="currentColor" stroke-width="1.5" stroke-linecap="round"/>
      <circle cx="6" cy="3.5" r="0.75" fill="currentColor"/>
    </svg>
    <span class="lbl-tooltip-popup">Your EU VAT registration number</span>
  </span>
</label>
