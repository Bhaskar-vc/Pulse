Alert

Alerts display important messages that inform users about system states, feedback, or required actions. They support semantic variants, dismissal, and contextual actions.

Stable WCAG AA Figma 230-4914

Overview

All four semantic variants displayed together. Each communicates a different intent through color, icon, and copy.

Do's & Don'ts

✓
Do
  • Use the correct semantic type (don't use info for errors)
  • Keep messages short and actionable
  • Use role="alert" for errors to announce immediately
  • Provide a single, clear action if needed
  • Place inline alerts near the content they describe
  • Auto-dismiss success toasts after 4–6 seconds
✕
Don't
  • Don't stack more than 3 alerts at once
  • Don't use error styling for warnings or neutral messages
  • Don't make critical error alerts dismissible without resolution
  • Don't put long paragraphs in alerts — link out instead
  • Don't use alerts for loading or progress states
  • Don't auto-dismiss error alerts

Accessibility

ARIA Roles

  • Use role="alert" for error alerts — announces immediately to screen readers
  • Use role="status" for success and info — polite announcement
  • Use role="status" for warning unless action is required

Color & Icons

  • Never use color alone to convey meaning — always pair with an icon
  • All color combinations meet WCAG AA 4.5:1 contrast ratio
  • Icons are decorative (aria-hidden="true"); meaning is in the text

Keyboard

  • Dismiss button must be reachable by keyboard (Tab)
  • Dismiss button activated with Enter or Space
  • After dismissal, focus should move to a logical next element

Motion

  • Dismiss animation respects prefers-reduced-motion
  • Auto-dismiss toasts pause on hover and keyboard focus
  • Animations are <300ms to avoid distraction

Code

alert.html — Basic dismissible alert 
<!-- Error alert with dismiss -->
<div class="alert alert-error"
     role="alert">

  <!-- Icon (aria-hidden) -->
  <div class="alert-icon" aria-hidden="true">
    <svg>...</svg>
  </div>

  <!-- Content -->
  <div class="alert-body">
    <div class="alert-title">
      Payment failed
    </div>
    <div class="alert-message">
      Please update your payment method.
    </div>
  </div>

  <!-- Dismiss -->
  <button
    class="alert-dismiss"
    aria-label="Dismiss alert"
    onclick="this.closest('.alert').remove()">
    ✕
  </button>
</div>
