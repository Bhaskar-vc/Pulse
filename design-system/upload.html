File Upload

A drag-and-drop upload zone for accepting file input. Supports single and multiple files, image previews, upload progress, and error states.

Stable WCAG AA Figma 232-209

Overview

Drag files into the zone or click to open a file picker. Files appear in the list below with upload progress, then settle into their success state.

Do's & Don'ts

✓
Do
  • Display accepted file types and max size in the zone
  • Show per-file upload progress feedback
  • Provide image thumbnails for visual file recognition
  • Allow removing files from the queue before submission
  • Label the zone clearly for screen readers
  • Validate on drop, not just on submit
✕
Don't
  • Don't accept files silently — always confirm receipt
  • Don't use upload to replace a simple file input where drag isn't relevant
  • Don't auto-submit on drop without user confirmation
  • Don't display file paths — just names and sizes
  • Don't block the UI during upload — show progress inline
  • Don't silently fail — always show clear error messages

Accessibility

Keyboard Access

  • Zone is reachable via Tab key (tabindex="0")
  • Enter or Space activates the file picker
  • Hidden file input is paired with a visible label
  • Remove buttons are fully keyboard-accessible

Screen Readers

  • Zone has role="button" and descriptive aria-label
  • Error messages use role="alert" for immediate announcement
  • Success confirmation should update a live region
  • File list items have accessible names including file name and size

Color Contrast

  • Dragover purple border (#7f56d9) passes AA contrast on white
  • Error red (#f04438) text on light background (#fef3f2) meets AA
  • Upload text colors (#29294c, #707087) both pass 4.5:1

Motion

  • File-in animation is subtle (180ms)
  • Progress bar transitions use ease — not jarring
  • Respects prefers-reduced-motion by skipping animations

Code

upload.html — Default drop zone 
<!-- File Upload Zone -->
<label for="upload">

  <div class="upload-zone"
       role="button"
       tabindex="0"
       aria-label="Upload files">

    <!-- Hidden real input -->
    <input type="file"
           id="upload"
           class="upload-input"
           multiple
           accept="image/*,.pdf"
           aria-hidden="true"
           tabindex="-1"/>

    <!-- Icon -->
    <div class="upload-icon">...</div>

    <div class="upload-text">
      <strong>Click to upload</strong>
      or drag and drop
    </div>

    <div class="upload-subtext">
      SVG, PNG, JPG up to 5MB
    </div>
  </div>
</label>
CSS — Core upload tokens 
/* Empty / Idle state */
.upload-zone {
  border: 2px dashed #eaeaed;
  border-radius: 12px;
  padding: 32px;
  background: #ffffff;
  text-align: center;
}

/* Drag over / active */
.upload-zone.dragover {
  border-color: #7f56d9;
  background: #f9f5ff;
}

/* Icon states */
.upload-icon      { color: #9d9dad; } /* idle */
.dragover .upload-icon { color: #7f56d9; } /* active */
