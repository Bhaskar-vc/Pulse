Slider

Sliders allow users to select a value or range from a continuous or stepped spectrum. Use them for settings like volume, brightness, price filters, or any numeric preference.

Stable Accessible Figma Ready

Overview

The slider wraps a native <input type="range"> with custom thumb and track styling. The filled track updates in real-time via a CSS linear-gradient background set from JavaScript.

Code

slider.html 
<!-- Slider with live value -->
<div class="slider-wrap">
  <div class="slider-label-row">
    <label class="slider-label" for="vol">Volume</label>
    <span class="slider-value" id="vol-val">65%</span>
  </div>
  <input
    class="slider"
    id="vol"
    type="range"
    min="0" max="100" value="65"
    oninput="updateSlider(this, 'vol-val', '%')"
  />
</div>

<!-- With steps -->
<input class="slider" type="range"
  min="1" max="12" step="1" value="6" />

<!-- Disabled -->
<input class="slider" type="range" disabled />
slider.js 
// Updates the live value readout and fills the track
function updateSlider(input, valId, suffix, prefix) {
  // Update readout
  const out = document.getElementById(valId);
  if (out) out.textContent = (prefix || '') + input.value + (suffix || '');

  // Fill track left of thumb
  const pct = ((input.value - input.min) / (input.max - input.min)) * 100;
  const color = input.dataset.color || '#7f56d9';
  input.style.background =
    `linear-gradient(to right, ${color} 0%, ${color} ${pct}%, #eaeaed ${pct}%, #eaeaed 100%)`;
}

// Init all sliders on page load
document.querySelectorAll('.slider').forEach(s => {
  updateSlider(s, null, '', '');
});

Accessibility

Keyboard

  • Arrow keys move the thumb by one step
  • Home / End jump to min / max
  • Page Up / Page Down jump by 10% of range

ARIA

  • Native <input type="range"> exposes role="slider" automatically
  • Always link with <label for="...">
  • Provide aria-valuetext when the raw number isn't self-explanatory (e.g. "Medium")

Visual

  • The track fill and thumb must have sufficient contrast
  • Do not rely on color alone — always show a numeric readout

Disabled state

  • Use the disabled HTML attribute — avoids pointer events automatically
  • Pair with a hint explaining why the slider is locked
