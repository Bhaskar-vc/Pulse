Do's & Don'ts

✓
Do
  • Use clear, concise column headers
  • Right-align numeric data for easy scanning
  • Use status badges consistently with the semantic color system
  • Provide an empty state with a helpful message
  • Show pagination for more than 10 rows
  • Keep actions minimal — 1–2 per row maximum
✕
Don't
  • Don't truncate critical data without a tooltip
  • Don't use more than 7–8 columns without horizontal scroll
  • Don't mix text and number alignment in the same column
  • Don't use decorative row colors that conflict with status badges
  • Don't make non-interactive rows look clickable
  • Don't nest tables inside table cells

Accessibility

Semantic HTML

  • Use role="table" on <table>
  • Use scope="col" on all <th> headers
  • Use scope="row" on row headers if applicable
  • Wrap in <thead>, <tbody>, <tfoot>

Sorting

  • Add aria-sort="ascending" or "descending" to active sort header
  • Non-sorted columns use aria-sort="none"
  • Sortable headers must be keyboard-focusable and activatable with Enter/Space

Selection

  • Checkbox inputs must have aria-label
  • Selected rows should announce state to screen readers
  • Select-all checkbox communicates indeterminate state with aria-checked="mixed"

Loading & Status

  • Add aria-busy="true" on table when loading
  • Empty state container should use role="status"
  • Pagination should use aria-label for prev/next buttons

Code

table.html — Basic striped table 
<!-- Striped Data Table -->
<div class="table-container">
  <table class="data-table striped"
         role="table"
         aria-label="Users">
    <thead>
      <tr>
        <th scope="col"
            class="sortable"
            aria-sort="ascending">
          Name
        </th>
        <th scope="col">Status</th>
        <th scope="col" class="num">Revenue</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Alice Chen</td>
        <td>
          <span class="badge badge-success">
            Active
          </span>
        </td>
        <td class="num">$4,200</td>
      </tr>
    </tbody>
  </table>
</div>
CSS — Core token values 
/* Table header */
.data-table thead th {
  background: #f9fafb;
  font-size: 11px;
  font-weight: 600;
  text-transform: uppercase;
  letter-spacing: 0.06em;
  color: #707087;
  border-bottom: 1px solid #eaeaed;
}

/* Row hover */
.data-table tbody tr:hover {
  background: #f4f4f6;
}

/* Selected row */
.data-table tbody tr.selected {
  background: #f9f5ff;
}
